## Using the Docked Chat

The docked chat lets you interact with the AI assistant while using other features.

### Opening the Docked Chat

- Click the **chat bubble icon** in the top header bar
- A chat panel will slide in from the right side
- Click the icon again to close it

### When to Use Docked Chat

The docked chat is useful when you want to:

- Ask questions while reading a transcription
- Get AI help while working on a translation
- Look up information without leaving your current page

### Differences from Full Chat

- The docked chat shares the same conversation as the full Chat page
- It has a narrower layout optimized for side-by-side use
- File uploads and all AI capabilities work the same way

### Tips

- The docked chat is hidden on mobile — use the full Chat page instead
- On the Write page, the editor has its own built-in chat panel
- Your docked chat position (open/closed) is remembered as you navigate
