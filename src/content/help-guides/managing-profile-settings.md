## Managing Profile Settings

The personalization portal keeps account, assistant, memory, connector, skill, and secret settings in one place.

### Open Settings

- Click your **profile avatar** → **Settings**
- Or type `/settings` in chat

### Sections

- **Discover** gives quick links to the main settings areas
- **Profile** manages your profile picture, language, and theme
- **AI assistant** manages the assistant name, model, and reasoning level
- **Memory** controls what Concierge remembers and whether it can update memory
- **Capabilities** manages connectors, skills, and secrets

### Secrets

Open **Settings** -> **Capabilities** -> **Secrets** to add, update, or delete API keys and tokens.

Saved secrets are encrypted and shown as masked password values. To replace a saved value, focus the value field, enter the new value, and click **Save**. After the save is verified, the field returns to a masked value and shows a saved confirmation.

### Tips

- Use **Connectors** to connect external tools and services
- Use **Skills** to save reusable instructions and supporting files
- Secret names must use `UPPER_SNAKE_CASE`
- Existing secret values are not revealed after they are saved
- Delete a secret only when you no longer want it available to your workspace
