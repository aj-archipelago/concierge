## Managing Profile Settings

Profile settings control how Concierge appears and how the assistant adapts to you.

### Open Settings

- Click your **profile avatar** and choose **Settings**
- Or type `/settings` in chat

### Common Settings

- **Profile** manages your profile picture, language, and theme
- **AI assistant** manages the assistant name, model, and reasoning level
- **Memory** controls what Concierge remembers and whether it can update memory

### Tips

- Use a clear assistant name if multiple team members share screenshots
- Review memory settings when Concierge starts giving stale context
- Keep language and theme preferences aligned with how you work most often
