## اختصارات لوحة المفاتيح ونصائح

### اختصارات الدردشة

| الاختصار        | الإجراء                        |
| --------------- | ------------------------------ |
| `Enter`         | إرسال الرسالة                  |
| `Shift + Enter` | إضافة سطر جديد                 |
| `/`             | فتح قائمة أوامر الشرطة المائلة |
| `/settings`     | فتح الإعدادات                  |

### نصائح للإنتاجية

- استخدم التطبيقات الصغيرة للمهام المتكررة
- ارفع الملفات مباشرة في الدردشة عندما تريد من Concierge تحليلها أو تلخيصها
- استخدم الشريط الجانبي للتنقل بين الدردشة والتطبيقات الصغيرة والوسائط ومساحات العمل
