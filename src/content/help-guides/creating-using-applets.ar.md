## إنشاء التطبيقات الصغيرة واستخدامها

التطبيقات الصغيرة أدوات تفاعلية تعمل داخل Concierge. يمكنها توفير واجهة HTML خاصة بها، واستخدام ملفات محددة النطاق، وتخزين حالة خاصة أو مشتركة، واستدعاء نماذج الذكاء الاصطناعي المسموح بها عبر حزمة Concierge Applet SDK.

### العثور على التطبيقات الصغيرة

1. افتح **التطبيقات الصغيرة**
2. ابحث باسم التطبيق عندما تكون القائمة طويلة
3. اختر تطبيق لوحة لفتحه في جلسة لوحة مرتبطة بدردشة جديدة
4. اختر تطبيقا مرتبطا بمساحة عمل لفتح مساحة العمل الخاصة به

### إنشاء تطبيق صغير

1. من **التطبيقات الصغيرة** اختر **إنشاء تطبيق صغير**
2. صف التطبيق الذي تريد من Concierge توليده
3. ينشئ Concierge دردشة محفوظة لجلسة التطبيق
4. راجع HTML الناتج واستمر في التعديل عبر الدردشة

استخدم **إنشاء مساحة عمل** في صفحة التطبيقات الصغيرة عندما تحتاج إلى منشئ التطبيقات الأقدم المرتبط بمساحات العمل.

### ملفات التطبيقات الصغيرة

تستخدم التطبيقات الصغيرة تخزينا محدد النطاق حتى يتمكن كل تطبيق من قراءة ملفاته وكتابتها من دون تصفح ملفات مستخدم غير مرتبطة. تستمر ملفات تطبيقات مساحات العمل الحالية في العمل عبر مسارات مساحات العمل القديمة، بينما تستخدم تطبيقات اللوحة الجديدة مسارات تخزين التطبيقات الصغيرة.

### Concierge Applet SDK

تتوفر حزمة SDK داخل HTML التطبيق باسم `window.ConciergeSDK`.

- استخدم `ConciergeSDK.agent.chat()` عندما يحتاج التطبيق إلى سياق المساعد الحالي أو أدواته
- استخدم `ConciergeSDK.models.generate()` لاستدعاءات النموذج المباشرة مثل التصنيف أو الاستخراج أو إعادة الصياغة أو التقييم أو توليد JSON
- استخدم `ConciergeSDK.files` لعرض ملفات التطبيق أو رفعها أو جلبها أو حذفها
- استخدم `ConciergeSDK.data` لإعدادات أو تقدم خاص بالمستخدم الحالي
- استخدم `ConciergeSDK.sharedData` لحالة مشتركة يجب أن يراها عدة مستخدمين للتطبيق نفسه

تجنب استدعاء واجهات SDK للذكاء الاصطناعي أو البيانات أو الملفات داخل حلقات عرض ضيقة أو سلاسل جلب مسبق متكررة أو مؤقتات غير محدودة.

### النشر

تتوفر مسارات التطبيقات المنشورة لعرض التطبيقات العامة. أبق المسودة والتطبيق المنشور منفصلين: حدّث المسودة أثناء التكرار، ثم انشر فقط الإصدار الذي يجب أن يراه المستخدمون الآخرون.
