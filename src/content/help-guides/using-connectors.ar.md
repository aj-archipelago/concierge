## استخدام الموصّلات

تتيح الموصّلات لـ Concierge استخدام أدوات وبيانات خارجية عبر خوادم Model Context Protocol (MCP) التي تضيفها إلى حسابك.

### فتح الموصّلات

- افتح محادثة
- اختر أيقونة القابس في شريط أدوات الدردشة
- راجع خوادم MCP المخصصة المتصلة بحسابك

### إضافة خادم MCP مخصص

- اختر **إضافة خادم**
- أدخل اسما ورابط خادم MCP
- ألصق رمز Bearer إذا كان الخادم يستخدم مصادقة بالرمز
- اترك حقل الرمز فارغا واختر **إضافة وربط OAuth** عندما يدعم الخادم OAuth

### إدارة الموصّلات

- تصبح الخوادم المتصلة متاحة للدردشة بعد إضافتها
- استخدم **ربط OAuth** أو **إعادة ربط OAuth** عندما يحتاج الخادم إلى تفويض
- أزل الموصّل عندما لا تريد أن يستخدم Concierge ذلك الخادم

### ملاحظات

- يحظر Concierge روابط الخوادم المخصصة غير الآمنة مثل روابط الشبكات الخاصة أو localhost أو link-local أو metadata
- يجب أن تستخدم روابط الخوادم المخصصة في الإنتاج HTTPS ما لم يتم السماح صراحة بالروابط الخاصة للتطوير المحلي
- تُخزّن الرموز وبيانات التحديث على الخادم ولا تُعاد إلى المتصفح
