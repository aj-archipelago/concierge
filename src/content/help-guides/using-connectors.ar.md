## استخدام الموصلات

تربط الموصلات Concierge بأدوات خارجية ليتمكن الذكاء الاصطناعي من البحث في بيانات تلك الخدمات وقراءتها والتفاعل معها مباشرة من المحادثة.

### فتح الموصلات

- اضغط على **صورة ملفك الشخصي** ← **الإعدادات** ← **القدرات** ← **الموصلات**
- أو اكتب `/connectors` في حقل المحادثة كاختصار

### الموصلات المتاحة (الاتصال السريع)

| الموصل                          | ما يتيحه                                                          |
| ------------------------------- | ----------------------------------------------------------------- |
| **Atlassian JIRA & Confluence** | البحث في القضايا، إنشاء التذاكر، قراءة وتحديث صفحات Confluence    |
| **Slack**                       | البحث في الرسائل والقنوات والمستخدمين في مساحة عملك (للقراءة فقط) |
| **GitHub**                      | تصفّح المستودعات والقضايا وطلبات السحب والتعليمات البرمجية        |

تستخدم هذه الإعدادات المسبقة OAuth: اضغط **اتصال**، أكمل تسجيل الدخول في نافذة المتصفح، ثم عُد إلى Concierge عند المطالبة. قد لا تظهر بعض الإعدادات المسبقة إلا بعد أن يفعّل المسؤول إعدادات OAuth المطلوبة في بيئتك.

### التطبيقات الصغيرة: استدعاء Jira REST API مباشرة

يمكن للتطبيقات الصغيرة استخدام `ConciergeSDK.services.getAccessToken({ service: "atlassian" })` ثم استدعاء Jira Cloud REST API باستخدام `metadata.baseUrl` ورأس `Authorization` الناتجين. عند البحث بـ **JQL**، استخدم **enhanced JQL search** على `/rest/api/3/search/jql` (`GET`، أو `POST` مع JSON للاستعلامات الأطول). لا تستخدم واجهة `/rest/api/3/search` القديمة بـ `GET` أو `POST`؛ فقد تمت إزالتها وتعيد **410**. راجع مرجع **[Issue search](https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-issue-search/)** وإشعار الترحيل **[CHANGELOG-2046](https://developer.atlassian.com/changelog/#CHANGE-2046)** قبل تنفيذ البحث حتى يبقى الكود مواكبًا.

### خوادم MCP مخصصة (أي رابط)

يمكنك إضافة **أي** خادم MCP يتحدث HTTP (HTTP قابل للبثّ)، وليس فقط الإعدادات المسبقة أعلاه.

1. افتح حوار الموصلات (`/connectors`)
2. تحت **الخوادم المخصصة**، اضغط **إضافة خادم**
3. أدخل **اسمًا** (يظهر في **موصلاتك** كما كتبته بالضبط)
4. أدخل **رابط MCP** — العنوان الكامل (يجب أن يبدأ بـ `http://` أو `https://`)
5. اختر طريقة المصادقة:
    - الصق **رمز Bearer** إذا كان الخادم يوفّره
    - أو اترك حقل الرمز فارغًا واضغط **إضافة وربط OAuth** لاستخدام OAuth 2.1
6. أكمل نافذة OAuth المنبثقة إذا ظهرت، أو اضغط **إضافة** لحفظ الخادم دون OAuth

يقوم Concierge أيضًا بإنشاء معرّف داخلي من الاسم (بأحرف صغيرة، وتُحوَّل الأحرف غير الأبجدية الرقمية إلى شرطات)، مثل `custom-my-server`. إذا أعدت استخدام اسم، تُضاف لاحقة رقمية (`custom-my-server-2`، إلخ) بحيث لا يتم استبدال شيء.

عند استخدام OAuth مع خادم مخصص، يكتشف Concierge بيانات OAuth الوصفية للخادم، ويسجّل نفسه ديناميكيًا للحصول على معرّف عميل، ثم يستخدم PKCE لتبادل رمز التفويض. يجب أن يوفّر الخادم بيانات OAuth القياسية ونقطة نهاية لتسجيل العملاء ديناميكيًا.

**الأمان:** أضف فقط خوادم MCP التي تثق بها. فضّل **HTTPS**. تعامل مع الرموز كأنها كلمات مرور.

### الاتصال بإعداد مسبق (الاتصال السريع)

1. افتح **الإعدادات** ← **القدرات** ← **الموصلات**
2. ابحث عن الخدمة تحت **الاتصال السريع**
3. اضغط **اتصال** وأكمل OAuth في النافذة المنبثقة أو إعادة التوجيه
4. إذا طلب أي إعداد مسبق رمزًا في الحوار، فألصقه واضغط **حفظ** (اتبع الرابط على الشاشة لـ "الحصول على رمز" حين يظهر)
5. تظهر الخدمات المتصلة تحت **موصلاتك** مع حالة (متصل، رمز منتهي الصلاحية، إلخ)

### استخدام موصل في المحادثة

بمجرد الاتصال، اطلب من Concierge بشكل طبيعي:

- "ابحث عن جميع تذاكر JIRA المفتوحة المسندة إليّ"
- "ابحث في Slack عن رسائل حول تقرير الربع الثاني"
- "أرني طلبات السحب المفتوحة في مستودع design-system"

سيستخدم Concierge الخدمة المتصلة لجلب بيانات حقيقية وتضمينها في إجابته.

### قطع اتصال الموصل

- افتح **الإعدادات** ← **القدرات** ← **الموصلات**
- ابحث عن الموصل تحت **موصلاتك**
- اضغط على **أيقونة سلة المهملات** لإزالته

### نصائح

- الموصلات مرتبطة بحسابك وتستمر عبر الجلسات
- إذا أظهر موصل **إعادة الاتصال** أو حالة منتهية، افتح الموصلات واتصل مرة أخرى؛ لتغيير الرمز لخادم مخصص، أزله ثم أضفه مرة أخرى بالرمز الجديد
- يمكن لخوادم MCP المخصصة استخدام رمز Bearer يدوي أو OAuth 2.1 مع تسجيل العميل الديناميكي
- تستخدم الموصلات [بروتوكول سياق النموذج (MCP)](https://modelcontextprotocol.io) في الخلفية
