## Using Skills

Skills give the AI assistant specialized knowledge and instructions for specific tasks. You can create your own skills or manage existing ones to shape how Concierge responds.

### Opening the Skills Manager

- Click your **profile avatar** → **Settings** → **Capabilities** → **Skills**
- Or type `/skills` in the chat input as a shortcut

### What Skills Do

Each skill contains a set of instructions that are loaded into the chat when activated. Skills can include:

- Custom instructions or persona adjustments
- Domain-specific knowledge and guidelines
- Supporting files (documents, references, templates)

### Creating a Skill

1. Open **Settings** → **Capabilities** → **Skills** and click **New Skill**
2. Give the skill a **name** and optional **description**
3. Write the skill instructions in the content editor
4. Optionally attach supporting files (PDFs, documents, etc.)
5. Click **Save**

### Using a Skill in Chat

- Open **Settings** → **Capabilities** → **Skills** and click the skill you want to activate
- The skill's instructions are loaded into the current chat session
- The AI will follow those instructions for the rest of the conversation

### Editing and Deleting Skills

- In the Skills Manager, click the **edit icon** next to a skill to update its name, description, or content
- Click the **delete icon** to remove a skill permanently

### Tips

- Skills are personal — they are only visible to you
- A skill can include reference files to give the AI access to specific documents
- Use skills for repeated tasks where you always want the same style or behavior (e.g. a writing tone, a coding style guide, a research template)
