## Translating Content

Concierge provides AI-powered translation for text, documents, and video content.

### Text Translation

1. Go to **Translate** from the sidebar
2. Enter or paste the text you want to translate
3. Select your target language
4. Choose the translation model that best fits the job, such as the newest OpenAI, Google, or Anthropic option, a faster model, or Azure
5. Click **Translate** to get the result

### Document Translation

- Upload a document file (PDF, DOCX, etc.) for translation
- The translated output preserves the original formatting where possible

### Video Translation

- For video content translation (subtitles, dubbing), use the **Transcribe** feature first to generate a transcript
- Background translation tasks appear in the **notifications** panel (bell icon)
- You'll be notified when the translation is complete

### Tips

- For best results, provide clean, well-formatted source text
- You can copy the translated text directly or download it
- Translation quality varies by language pair — review outputs for critical content
