## Getting Started with Concierge

Concierge is an AI assistant for chat, file work, translation, transcription, writing, and custom applets.

### Navigation

Use the sidebar to move between core areas:

- **Chat** — Have conversations with the AI assistant
- **Translate** — Translate content between languages
- **Transcribe** — Convert audio and video to text
- **Write** — Draft and edit content with AI assistance
- **Applets** — Use and create custom AI-powered mini-apps
- **Media** — Manage uploaded files

### Quick Tips

- Click your **profile avatar** → **Settings** to adjust model, language, theme, and memory preferences
- Type **`/`** in a chat input to see available commands
- Use notifications to check background tasks
- Use **Send feedback** in the sidebar to report an issue, share an idea, or ask a question
