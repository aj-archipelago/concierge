## Keyboard Shortcuts and Tips

### Chat Shortcuts

| Shortcut        | Action                      |
| --------------- | --------------------------- |
| `Enter`         | Send message                |
| `Shift + Enter` | Add a new line              |
| `/`             | Open the slash-command menu |
| `/settings`     | Open settings               |

### Productivity Tips

- Use applets for repeatable tasks
- Upload files directly in chat when you want Concierge to analyze or summarize them
- Use the sidebar to switch between Chat, Applets, Media, and workspace flows
