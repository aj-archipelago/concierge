## Keyboard Shortcuts & Tips

Speed up your workflow with these keyboard shortcuts and productivity tips.

### Chat Shortcuts

| Shortcut        | Action                            |
| --------------- | --------------------------------- |
| `Enter`         | Send message                      |
| `Shift + Enter` | New line in message               |
| `/`             | Open slash command menu           |
| `/settings`     | Open the personalization portal   |
| `/connectors`   | Jump to Capabilities → Connectors |
| `/skills`       | Jump to Capabilities → Skills     |

### General Tips

- **Settings** — Click your profile avatar → **Settings** to personalize Concierge (model, theme, language, memory, connectors, skills)
- **Notifications** — Click the bell icon to check on background tasks
- **Quick chat** — Use the chat icon in the header to open a docked chat panel on any page

### Productivity Tips

- Use **applets** for repetitive tasks — create a custom applet once, reuse it forever
- **Upload files** directly in chat for quick AI analysis
- Check the **Home page** for a digest of recent activity
- Use **Translate** for quick text translations instead of asking in chat for better results
